# Music

Drop royalty-free background music here — the pipeline picks one track at random and mixes it under the voiceover at 12% volume.

- Use instrumental, low-intensity tracks (ambient, lo-fi, cinematic pads).
- Free sources: YouTube Audio Library, Pixabay Music, Mixkit (check licenses).
- Supported: `.mp3 .m4a .wav .ogg .flac`

If this folder is empty, videos render with voiceover only — which is fine; many faceless channels skip music entirely, and platforms let you add trending sounds at upload time (often better for reach).
