# Backgrounds

Drop background videos or images here — the pipeline picks one at random per video, scales and crops it to 1080×1920, and loops it for the full runtime.

- Best: vertical (9:16) ambient loops — slow drone shots, abstract motion, space, ocean, city timelapses.
- Good free sources: Pexels Videos, Pixabay, Mixkit (check each item's license; all three offer free-for-commercial-use content).
- Supported: `.mp4 .mov .mkv .webm .avi .jpg .jpeg .png .webp`

If this folder is empty, the pipeline generates an animated gradient in the scenario's brand colors instead — works fine, but real footage holds attention better.
